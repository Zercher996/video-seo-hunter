整条路径只有 **5步**，唯一的外部依赖就是 YouTube API，没有后端，没有数据库，极简。



关键实现细节：
- 切词用 `title.split(' ').filter(Boolean)`，去掉多余空格
- 点词组合：选中的词用空格拼接，形成一个关键词字符串
- YouTube 搜索参数：`videoDuration=long` + `order=viewCount` + `type=video`，每次取 `maxResults=3`
- API Key 存在 `.env` 的 `VITE_YOUTUBE_API_KEY`，前端直接调用
